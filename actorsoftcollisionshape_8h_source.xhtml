52  /// @class ActorSoftCollisionShape
53  /// @headerfile collisionshape.h
54  /// @brief A collision shape for soft actors.
55  /// @details This collision shape is different from all the other collision shapes because it's
56  /// meant to be used for only one type of object: ActorSoft's. This shape doesn't use triangles
57  /// in the traditional manner other mesh shapes do, but rather it uses the individual internal
58  /// nodes that make the ActorSoft as points for building triangles to process collisions with. As
59  /// a result a SoftBodyCollisionShape should never be reused and should be kept unique to the Actor
60  /// it applies to. @n @n
61  /// Normally the ActorSoft will be responsible for the creation of it's own collision shape, so
62  /// the user should never have to. So there are no non-internal constructors provided for this class.
63  ///////////////////////////////////////

65  {
66  protected:
67  btSoftBodyCollisionShape* ActorSoftShape;
