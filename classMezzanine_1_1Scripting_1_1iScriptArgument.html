Detailed Description

The interface for a script argument.

All the members that all script for all languages must implement.

These are created to provide data to scripts. This is intended to primarily to act as an abstraction layer between game data and data inside scripts, but currently just transports primitives into and out of scripts without needing to care about any underlying types. This limits what operations can be done with this, because it is intended to provide an abstraction that can be used to implement specific algorithms for languages that require them.
