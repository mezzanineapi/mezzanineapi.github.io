246  /// @class NonStaticWorldObject
247  /// @headerfile worldobject.h
248  /// @brief This is a small extention of the WorldObject class that adds features common to non-static objects.
249  /// @details Many world objects are capable of being moved and oriented dynamically and at run time, but there
250  /// are some objects that you don't want to have the ability or API to do so as it may break the intent of the
251  /// object. This class is here to be avoided for those objects, and instead inherit directly from the
252  /// WorldObject class while all other's inherit from this class. It's important to note that this class doesn't
253  /// forbid static objects as the name may suggest, but rather it makes static status optional instead of forcing
254  /// them to be static.
255  ///////////////////////////////////////
