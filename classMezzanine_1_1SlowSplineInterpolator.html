Detailed Description

template<typename T>
class Mezzanine::SlowSplineInterpolator< T >

If something specifically needs the linear interpolator for T they should use this.

This is with be a cubic spline where applicable, and will be more smooth that the others, and be at least as intuitive as the linear version:

  • Data points, will be valid interpolated values.
  • There are no "Corners".
  • Execution time is O^N.
  • This shape defined by interpolating a set of these will leave a Convex Hull(or Axis Aligned Bounding Box) that could contain the data.
  • Will be able to interpolated arbitrary sets of data points.
