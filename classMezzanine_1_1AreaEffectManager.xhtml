Detailed Description

A manager responsible for the storage and management of all areaeffects in use.

More or less Management point for a container of areaeffects to help keep them sorted.

void Mezzanine::AreaEffectManager::AddAreaEffect ( AreaEffect *  ToBeAdded)
virtual

Adds a pre-created areaeffect to the manager.

In some cases you may want to add and remove an areaeffect from the world without destroying it and do some special manipulations to it to achieve some special/unique affects. This function along with the "RemoveActor()" function facilitates this.
This function is also necessary for anyone inheriting from our actors to add their actors to the world.

Parameters
ToBeAddedThe areaeffects to be added to the manager.
