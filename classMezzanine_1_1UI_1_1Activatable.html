void Mezzanine::UI::Activatable::EnableMultipleActivations ( bool  Enable)
virtual

Enables or disables whether or not the activatable should be allowed to activate multiple times per frame.

In most cases a activatable will only activate multiple times when using hotkeys, either when there are multiple keys hotkeyed to the same activatable, or when the mouse activatable is pressed over a UI activatable while a hotkey for it is activated in the same frame, or both. If you only want a UI element being triggered once per frame at most, you want this disabled.
Default: false.

Parameters
EnableWhether or not to enable this feature.
